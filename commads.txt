> git init [for initialising git]
> git status [list all untracked files ]
> git add -A [adding all untracked files to staging area] 
> git commit -m "commit message" [to commit added files to local repository]
> git clone [to take exact copy from remote repository to local repository]
> git pull [to take changes from remote repository to local work space ]
> git push [to send changes from local repository to remote repository]
> git config --list [to list configuration]
> git config --global user_name "username" [to add username]
> git config --global user_email "useremail" [to add useremail]
> git remote add remote_name remote_path [to add remote ]
> git push remote_name branch
